Overview

Purpose

I want to create this site for the sake of my DM, David, to help new players get used to his custom setting and homebrew (setting specific additions). I also want to display character information to help new players to better understand the setting.

Audience

This site is for new players to David's setting of Mezadria.

Dynamic elements

I want to store character info and descriptions in JSON files, then load the info into the website to be accessed through a dropdown menu. When the Javascript loads, it will read the JSON into an object array. It will require adding an EventListener that will wait for the user to select a character name from the dropdown menu. For the dropdown menu, it will also require its own EventListener for showing the menu. When a character name is selected the array will be filtered until only the characters that have matching names will have their information displayed through a template literal.
